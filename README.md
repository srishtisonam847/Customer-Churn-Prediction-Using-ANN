# Customer Churn Prediction using ANN

## Project Overview

Built an Artificial Neural Network using TensorFlow/Keras to predict whether a customer will leave the bank.

## Technologies Used

- Python
- Pandas
- NumPy
- TensorFlow
- Keras
- Scikit-learn
- Matplotlib

## Model

- Sequential ANN
- ReLU Activation
- Sigmoid Output
- Adam Optimizer
- Binary Crossentropy


## Accuracy

Accuracy: 86%

## Dataset

Bank Customer Churn Dataset
